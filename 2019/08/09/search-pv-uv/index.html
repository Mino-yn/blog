查询系统中的pv/uv数据

发表于 2019-08-09

连接至数据库

打开数据库管理系统，新建MySQL连接，输入对应的ip地址、用户名、密码、连接至数据库

查询pv/uv数据

进入matomo库中的matomo_site表查看需要查询数据的系统所对应的id号码（idsite）。

新建查询，将需要进行pv/uv数据查询的系统所对应的idsite填入，并修改日期范围。

导出查询结果

点击导出结果，选择导出当前结果，选择导出格式为Excel数据表进行导出。即可得到包含有系统pv/uv数据的Excel表格。
