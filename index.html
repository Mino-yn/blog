删除大文件，磁盘空间未释放

发表于 2020-05-18

背景

HDY 数据库所在服务器到期，需要腾出磁盘空间，进行数据库备份迁移，查看文件大小，发现有一个非常大的err日志文件 326GB, 为腾空间，决定将其删除。

过程

删除后查看磁盘空间，发现空间未释放。应该有程序还在操作这个文件，指针未释放

删除进程

总结

  • 方法一

    在删除之前 先清空文件，再删除文件 这样能快速释放磁盘空间。

  • 方法二

    若文件已删除，但磁盘空间未被释放，很可能是此时这个文件被别的进程所占用，kill 掉占用文件的进程号即可解决。注意：kill 进程之前，确认一下删除的是什么进程，防止删除了重要的进程，避免不必要的麻烦

PV UV IP

发表于 2020-04-23

简介

PV、UV、IP是在运营和网站管理中常用的概念

PV

页面浏览量：用户实际浏览过的网页数量

访问量，即页面浏览量或点击量，衡量网站用户访问的网页数量；在一定统计周期内用户每打开或刷新一个页面就记录1次，多次打开或刷新同一页面则浏览量累计

UV

独立访客，统计1天内访问某站点的用户数；访问网站的一台电脑客户端为一个访客，可理解成访问某网站的电脑数量。

IP

独立IP数，1天内多少个独立的IP浏览了页面，即统计不同的IP浏览用户数量。
